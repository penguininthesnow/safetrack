# Safetrack
第三階段個人專案

- 會員系統(User):
  - 一般作業人員: 查看個人資料、修改姓名、部門名稱
  - 主管: 查看個人資料、修改姓名、部門名稱、瀏覽所有會員資料與職位的修改

- 巡檢系統(Inspections):
  - 建立巡檢紀錄(並結合line messageing API做及時上傳回報)
  - 歷年巡檢紀錄查詢: 以篩選的方式進行搜尋(包括:巡檢年份、地點、項目以及異常與否)
  - 預覽/修改/刪除  巡檢紀錄
  - 統計圖表分析(圓餅圖、長條圖)

- 通知設定(Notification_setting):
  - 使用權限:主管
  - 說明: 異常通啟用與否、發送的群組名稱設定、發送的群組ID設定
